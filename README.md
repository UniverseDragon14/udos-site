# Universal Dragon Operating System (UDOS)

UDOS is a long-term Universal Dragon concept by Aslam.

It is not a finished operating system yet.  
Current real direction: assistant-first control layer + EVE/NOVA brain + safe tools + memory + rollback + future hardware.

## Core Idea

Phone is not app-based.  
Phone is assistant-based.

UDOS is a future control layer where EVE/NOVA helps manage:
- user commands
- memory
- files
- voice
- Pi5
- devices
- printer
- camera
- sound
- future robot/hardware modules

## Current Real Status

- EVE Mobile UI prototype exists
- Moto G22 is used as safe test phone
- Raspberry Pi 5 is used as brain/server/build device
- Tailscale remote access is active
- EVE web UI can run from Pi
- Replit EVE Brain is working
- Tamil reply test works
